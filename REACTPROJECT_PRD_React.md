**PRD: 대화 기반 AI 독서치료 시스템**

**바이오헬스 × 생성형 AI × LangChain 융합 플랫폼**

⸻

**1. 제품 개요 (Product Overview)**

본 시스템은 사용자의 감정 상태를 기반으로 맞춤형 대화 상담을 제공하고, 그에 맞는 책을 AI가 추천하는 **대화 기반 독서치료(Book Therapy) 웹 서비스**이다.

연령대별 상담 톤, 감정 분석, 독서 기록 기능을 결합하여 개인별 맞춤 독서치료 경험을 제공한다.

⸻

**2. 목표 (Goals)**

•	감정 분석 기반 독서 추천

•	연령대별 최적화된 대화 경험 제공

•	대화 히스토리 기반 회상·성찰 기능 강화

•	독서 기록(서재) 관리 및 감정 변화 시각화

•	LangChain 기반 문맥 기억 및 분석

⸻

**3. 핵심 기능 (Key Features)**

**3.1 로그인**

•	이름 입력 → 간단 로그인

•	사용자 정보 저장(LocalStorage 또는 DB)

⸻

**3.2 모드 선택**

•	선택 가능한 모드:

•	**어린이 / 청소년 / 어른 / 노인**

•	모드별로 말투, 대화 스타일, 답변 톤 변경

•	UI도 모드별 분위기 적용 가능

⸻

**3.3 홈 화면**

•	프로필 이미지

•	사용자 이름 표시

•	출석률(일간 접속 기록)

•	주요 버튼:

•	**서재**

•	**대화하기**

•	간단한 AI 인사 메시지 표시

⸻

**3.4 서재(Bookshelf)**

•	추천/읽은 책을 표지 이미지 형태로 진열

•	실제 책장 같은 카드형 UI

•	책 클릭 시 → **독서기록장(Book Log)** 이동

⸻

**3.5 독서기록장(Book Log)**

독서기록장은 책을 클릭하면 확인할 수 있는 “AI 감정 요약 + 사용자의 소감 기록” 페이지이다.

**구성 요소**

**1) 책 기본 정보**

•	표지 이미지

•	제목

•	저자

•	추천 날짜

**2) AI 감정 요약(AI Emotion Summary)**

AI가 책을 추천했던 당시 사용자 대화를 분석하여 생성한 감정 요약.

예시:

### AI 감정 요약

- 주요 감정: 불안 72%
- 대화 중 사용자의 고민: “나는 요즘 선택이 너무 어렵다…”
- 핵심 키워드: 선택, 지침, 불안
- 추천 이유: 안정감을 주는 서사가 있어 심리적 회복에 도움됩니다.

**3) 사용자의 독서 소감(User Review)**

사용자가 책을 읽고 느낀 점을 남기는 자유 입력 칸.

예시 UI:

### 나의 독서 소감

(예: 이 책을 읽고 마음이 한결 편안해졌어요…)

**4) 기록 저장 기능**

•	입력된 소감을 DB에 저장

•	수정 가능 / 최종 수정 날짜 기록

**5) 감정 변화 시각화(선택)**

•	추천 당시 감정 vs 현재 감정 비교(그래프 형태)

⸻

**3.6 대화하기(Chat)**

•	GPT + LangChain 기반 AI 대화 서비스

•	실시간 감정 분석

•	감정 기반 + 키워드 기반 책 추천

•	모드별 말투/응답 스타일 자동 적용

⸻

**3.7 모드별 대화 스타일**

**어린이 모드**

•	반말, 귀여운 톤

•	정서적 안정 중심

•	예: “그랬구나! 힘들었겠다. 내가 도와줄게!”

**청소년 모드**

•	공감 + 현실 조언

•	비밀 친구 느낌

•	예: “그런 고민 진짜 힘들지. 근데 너라면 잘 해낼 수 있어.”

**어른 모드**

•	차분하고 성숙한 조언

•	심리·문제 해결 중심

•	예: “그 상황에서 느낀 감정은 자연스러운 것입니다. 함께 해결책을 생각해보죠.”

**노인 모드**

•	회상 대화 중심

•	느리고 따뜻한 표현

•	예: “예전에 비슷한 경험을 하셨을까요? 천천히 이야기해 주세요.”

⸻

**4. 사용자 스토리 (User Stories)**

**US-01**

사용자로서, 이름만 입력하여 간편하게 로그인하고 싶다.

**US-02**

사용자로서, 나의 나이에 맞는 대화 모드를 선택하고 싶다.

**US-03**

사용자로서, 홈에서 프로필 이미지와 출석률을 확인하고 싶다.

**US-04**

사용자로서, 지금까지 추천받은 책을 책장 형태로 보고 싶다.

**US-05**

사용자로서, 책을 클릭하면 AI가 남긴 감정 요약을 확인하고 싶다.

**US-06**

사용자로서, 책에 대한 내 감상을 자유롭게 기록하고 싶다.

**US-07**

사용자로서, 대화에서 내 감정을 분석받고 위로를 얻고 싶다.

**US-08**

사용자로서, 감정 상태에 맞는 책을 AI에게 추천받고 싶다.

**US-09**

노년층 사용자로서, AI와 회상 대화를 나누고 싶다.

**US-10**

사용자로서, 대화 기록이 나의 성장 기록으로 연결되길 원한다.

**5. 기술 구조 (Technical Architecture)**

**Frontend**

•	React

•	Vite

•	Tailwind CSS

•	Zustand / Recoil

**Backend**

•	Node.js + Express

•	또는 Serverless (Vercel Functions)

**AI**

•	ChatGPT API

•	LangChain 구성 요소

•	ConversationBufferMemory

•	감정 분석 Chain

•	추천 Chain

**Database**

•	Supabase / Firebase / PostgreSQL

•	저장 항목:

•	사용자 정보

•	대화 로그

•	감정 분석 결과

•	책 추천 내역

•	독서 기록(Book Log)

⸻

**6. 화면 구조 (Information Architecture)**

1. 로그인

- 이름 입력

2. 모드 선택

- 어린이 / 청소년 / 어른 / 노인

3. 홈

- 프로필 이미지

- 이름

- 출석률

- 서재 버튼

- 대화하기 버튼

4. 서재

- 책 이미지 목록

- 클릭 시 독서기록장 이동

5. 독서기록장(Book Log)

- 책 정보

- AI 감정 요약

- 사용자 소감 입력 칸

- 저장

6. 대화하기

- LangChain 대화 엔진

- 감정 분석

- 책 추천

- 모드별 톤 적용

⸻

**7. 감정 기반 추천 로직 (Emotion-driven Recommendation)**

1.	사용자 입력 수집

2.	LangChain이 대화 문맥 저장

3.	GPT로 감정 분석

4.	감정 스코어(불안, 우울, 기쁨 등) 추출

5.	감정-책 카테고리 매칭

6.	연령대 필터 적용

7.	1~3권 추천

8.	클릭 시 서재에 저장
